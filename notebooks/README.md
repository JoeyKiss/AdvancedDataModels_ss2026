# Notebooks Directory

This directory contains Jupyter notebooks used for demonstrations and exploratory analysis during the Advanced Data Models seminar.

Guidelines:

- Each notebook should be self-contained and clearly organized, with explanatory comments and markdown cells to describe the purpose of the code.
- Use open-source libraries and datasets that can be downloaded or accessed easily by other students.
- Keep dependencies minimal; if special packages are required, document them in the `requirements.txt` or the notebook's first cell.
- Ensure that any data used in the notebook is stored in the `/data` directory and loaded relative to that path.
- Include visualizations and outputs that help illustrate key concepts of your assigned data model.

Place your notebooks in this directory and reference them in your presentation materials.
